# Design: Brand, UX & Psychology

> **Last Updated:** January 16, 2026

## Overview

PropTraderAI's design philosophy is "Vibe-First, Power Later, Platform Last." We prioritize emotional resonance over feature density, especially in Phase 1.

**Core Principle:** PropTraderAI is not Headspace for traders. It's **AA sponsor meets sports psychologist meets risk manager**.

## Key Design Principles

### Phase 1: Vibe-Only Consumer App

1. **One thing at a time** ‚Äî ONE hero card, everything else scrolls
2. **Default to hiding complexity** ‚Äî If in doubt, don't show it
3. **Mobile-first always** ‚Äî Desktop is secondary
4. **Frame protection as $ saved** ‚Äî Not discipline, not willpower, MONEY
